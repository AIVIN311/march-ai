# Match AI / 授權式相遇系統 MVP v0.1

> 不是滑卡配對，而是先理解再介紹。

用授權式理解與低頻提案，讓相遇從一段真實開始。

這是一個手機優先的單頁互動原型，用來驗證兩件事：
1. 以「被理解」開始，是否比傳統交友流程更有感。
2. 被系統理解的感覺，能不能做到溫柔而不侵入。

## 這版做什麼

- 單頁式前端原型，使用原生 `HTML / CSS / JS`
- 日記式首頁、提案頁、任務交換頁、半自由對話頁四段主線
- 少量真輸入加固定規則，驅動後續摘要與 demo 文案
- 扶手式對話入口：
  - `延續剛才`
  - `換輕一點`
  - `再深一點`
  - `開一個小任務`
  - `今天先到這裡`

## 這版不做什麼

- 不接後端
- 不做登入
- 不做真實媒合
- 不接 AI API
- 不處理照片交換、通知、外部聯絡方式

## 四頁流程

### 1. 首頁

- 顯示日期、狀態句、理解摘要卡
- 可輸入一小段近況，立即更新理解摘要
- 顯示相遇收件匣與人格防火牆雛形

### 2. 提案頁

- 用介紹信語氣呈現來信提案
- 顯示提案理由、對方內在輪廓、節奏提醒
- 提供接受 / 稍後 / 拒絕三個分支

### 3. 任務交換頁

- 使用者先寫自己的回答
- 提交前看不到對方內容
- 揭露後可圈選一句有感的話並送出短回應

### 4. 半自由對話頁

- 保留自由輸入
- 同時提供五個扶手入口
- 用 demo 回覆模擬「慢慢接住」的聊天節奏

## 如何打開原型

直接用瀏覽器開啟 [index.html](./index.html) 即可。
