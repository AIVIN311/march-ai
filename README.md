# March / Match AI

> 不是滑卡配對，而是先理解，再介紹。

March 是品牌本體，Match AI 是產品描述。  
這是一個手機優先、單頁式的 dark-journal 互動原型，用來驗證一種更安靜的相遇方式：先讓使用者被理解，再讓介紹開始。

## Document Map

- **Start here:** [Project Brief](./docs/project-brief.md)
- **Philosophy:** [Constitution](./docs/match-ai-constitution-v0.1.md)
- **Product guardrails:** [Product Principles](./docs/match-ai-product-principles-v0.1.md)

## Current Prototype

- Single-page `HTML / CSS / JS` prototype
- Dark-journal visual direction
- Brand-first landing with inline onboarding v0.2
- English-first UI on Entry / Today / Proposal
- Reviewed-understanding flow before Today / Proposal
- Preview-only introduction mode when matching is turned off

## How To Open

Open [index.html](./index.html) in a browser.

## Current Focus

- Keep the onboarding calm, inline, and non-wizard-like
- Preserve the distinction between:
  - `your original words`
  - `your reviewed understanding`
  - `what can be used for introductions`
- Improve product feel before expanding system scope

## Not In Scope Yet

- Backend / auth
- Real matching engine
- Photo exchange / external contact sharing
- Notifications
- Live AI or agent infrastructure
